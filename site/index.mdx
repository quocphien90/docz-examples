# Getting Started

An UI design language and React-based implementation, created with [Styled-System](https://github.com/jxnblk/styled-system).

## Installation

### via yarn
```bash
yarn add audora-ui
```
### via npm
```bash
npm install audora-ui --save
```

## Using

```
import React from 'react'
import { ThemeProvider, Button } from 'audora-ui'

export default () => (
  <ThemeProvider>
    <Button>Press</Button>
  </ThemeProvider>
)
```
